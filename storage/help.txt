Список доступных комманд:

add       - добавление задачи. После ввода данной команды следует построчно ввести 
            поля: название, описание, дата. Поле 'дата' является составным, т.е. 
            при вводе отдельно указывается год, месяц, день и т.д
	
del       - удаление задачи. Для удаления задачи следует ввести название удаляемой задачи

edit      - построчное редактирование указанной задачи

search    - поиск задачи. Для поиска введите название задачи

show all  - вывести список всех задач из журнала

clear all - очистить журнал задач

record    - записать список задач на диск в xml формате в дефолтный файл или с указанным имененем

read      - считать список задач с диска. Укажите имя файла, иначе будет выбран дефолтный файл

stop      - остановить работу данного приложения

help      - справка по командам
